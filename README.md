# Simple Python To-Do List

This is a beginner-friendly To-Do List application built using Python. It helps users manage their daily tasks with a simple command-line interface.

## 🚀 Features
- **Add Tasks:** Quickly add new things to do.
- **Track Status:** Mark tasks as 'Pending' or 'Completed'.
- **Save Data:** Uses a JSON file to save your tasks so they don't disappear when you close the program.
- **Delete Tasks:** Remove unwanted tasks easily.

## 🛠️ Built With
- **Python:** Core logic and loops.
- **JSON Module:** For permanent data storage.

## 📝 How to Use
1. Make sure you have Python installed on your system.
2. Download the `python-todo-app.py` file.
3. Run the file using the command: `python python-todo-app.py`
4. Follow the menu options (1-5) to manage your tasks!
